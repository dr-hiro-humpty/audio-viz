# audio-viz

## 概要

electron + jsで音声の可視化を行います。

## 書いた人

濱矢 裕

## ライセンス

MITライセンス

## 動かし方

### 必要なもの

- node.js
- electron

### 実行

```
$ npm install --save-dev electron
$ npm run start
```
